- **不要追求完美**：跑通就算 70 分，有录屏算 80 分，有架构图 + 详细 README 算 90 分

- **如果编译 llama.cpp 卡住**：直接用 `MLCEngine`（封装更好，Android 集成更简单），不要浪费时间
